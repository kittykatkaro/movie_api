myFlix API Documentation

Overview

This API allows users to interact with the myFlix movie database. Users can view movies, genres, and directors, register as users, and manage their favorite movies list. Below are the routes and methods provided by the API.

GET /movies

Returns a list of all movies in the database.

http://localhost:8080/movies

Response:

[
  {
    "title": "Inception",
    "description": "A mind-bending thriller",
    "genre": { "name": "Sci-Fi", "description": "Science Fiction movies" },
    "director": { "name": "Christopher Nolan", "bio": "Famous director", "birth_year": "1970", "death_year": null },
    "imageUrl": "/path/to/image.jpg",
    "featured": true
  },
  {
    "title": "The Dark Knight",
    "description": "A thrilling superhero movie",
    ...
  }
]

This endpoint provides detailed data about each movie including the title, description, genre, director, image URL, and whether itâ€™s featured.

GET /movies/:title

Returns data about a single movie by title, including:

  • Description
  • Genre
  • Director
  • Image URL
  • Featured status
 http://localhost:8080/movies/Inception

Response:

{
  "title": "Inception",
  "description": "A mind-bending thriller",
  "genre": { "name": "Sci-Fi", "description": "Science Fiction movies" },
  "director": { "name": "Christopher Nolan", "bio": "Famous director", "birth_year": "1970", "death_year": null },
  "imageUrl": "/path/to/image.jpg",
  "featured": true
}

GET /genres/:name

Returns data about a specific genre by name.

 http://localhost:8080/genres/Action

Response:

{
  "name": "Action",
  "description": "Action-packed movies"
}

GET /directors/:name

Returns data about a director by name, including:

  • Bio
  • Birth year
  • Death year (if applicable)
 http://localhost:8080/directors/Nolan

Response:

{
  "name": "Christopher Nolan",
  "bio": "Famous director known for complex, mind-bending films",
  "birth_year": "1970",
  "death_year": null
}

POST /users

Allows new users to register by providing:

  • Username
  • Password
  • Email
  • Date of Birth
 POST http://localhost:8080/users -d '{"username":"johnDoe","Password":"1234","Email":"john@example.com","Birthday":"1990-01-01"}'

Response:

{
  "username": "johnDoe",
  "Password": "1234",
  "Email": "john@example.com",
  "Birthday": "1990-01-01"
}

PUT /users/:username

Allows users to update their user information including:

  • username
  • Password
  • Email
  • Date of Birth
PUT http://localhost:8080/users/johnDoe -d '{"Email":"johnnew@example.com"}'

Response:

{
  "username": "johnDoe",
  "Email": "johnnew@example.com",
  "Password": "1234",
  "Birthday": "1990-01-01"
}

PUT /users/:username/movies/:MovieID

Allows users to add a movie to their list of favorite movies by movie ID.

PUT http://localhost:8080/users/johnDoe/movies/5f8d0d55b54764421b7156c7

Response:

{
  "username": "johnDoe",
  "FavoriteMovies": ["5f8d0d55b54764421b7156c7"]
}

DELETE /users/:username/movies/:MovieID

Allows users to remove a movie from their list of favorite movies by movie ID.

DELETE http://localhost:8080/users/johnDoe/movies/5f8d0d55b54764421b7156c7

Response:

{
  "username": "johnDoe",
  "FavoriteMovies": []
}

DELETE /users/:username

Allows existing users to deregister by username.

DELETE http://localhost:8080/users/johnDoe

Response:

johnDoe was deleted.
