API Documentation

1. Get Top 10 Movies

High-Level Description: Returns a list of the top 10 movies in JSON format.

Endpoint URL: /movies

Method Type: GET

Query Parameters: None

Response Format:

JSON array containing movie objects.

  {
    "title": "The Godfather",

2. Get Movie by Title

High-Level Description: Retrieves a movie object by its title.

Endpoint URL: /movies/:title

Method Type: GET

Query Parameters:

  • title (required): The title of the movie.

Response Format:

JSON object containing the movie details.

{
  "title": "The Godfather",
  "description": "The aging patriarch of an organized crime dynasty transfers control of his clandestine empire to his reluctant son.",
  "imageURL": "https://upload.wikimedia.org/wikipedia/en/1/1c/Godfather_ver1.jpg",
  "director": {
    "name": "Francis Ford Coppola",

  },
  "genre": {
    "name": "Crime",
    "description": "Crime movies are movies that focus on criminal activities."
  }
}

3. Get Genre Description

High-Level Description: Returns the description of a specified genre.

Endpoint URL: /genres/:name

Method Type: GET

Query Parameters:

  • name (required): The name of the genre.

Response Format:

Plain text description of the genre.

"Crime movies are movies that focus on criminal activities."

4. Get Director Description

High-Level Description: Returns the description of a specified director.

Endpoint URL: /directors/:name

Method Type: GET

Query Parameters:

Response Format:

Plain text description of the director.

"Francis Ford Coppola is an American film director, producer, and screenwriter."

5. Create New User

High-Level Description: Creates a new user.

Endpoint URL: /users

Method Type: POST

Query Parameters: None

Body Data Format: Not specified in the code (could include user data like username, email, etc.).

Response Format:

7. Add Movie to User's Favorite List

High-Level Description: Adds a movie to a user's list of favorites.

Endpoint URL: /users/:username/movies/:title

Method Type: PUT

Query Parameters:

  • username (required): The username of the user.
  • title (required): The title of the movie to add.

Response Format:

Plain text message indicating success.

"Movie The Godfather added to johndoe's favorite list."

8. Remove Movie from User's Favorite List

High-Level Description: Removes a movie from a user's list of favorites.

Endpoint URL: /users/:username/movies/:title

Method Type: DELETE

Query Parameters:

  • username (required): The username of the user.
  • title (required): The title of the movie to remove.

Response Format:

Plain text message indicating success.

"Movie The Godfather removed from johndoe's favorite list."

9. Deregister User

High-Level Description: Deregisters (deletes) a user from the system.

Endpoint URL: /users/:username

Method Type: DELETE

Query Parameters:

  • username (required): The username of the user to deregister.

Response Format:

Plain text message indicating success.

"User johndoe deregistered successfully."
